# Jellyfin-RandomMovie

Simple web app for find a random movie from your Jellyfin Server.

![Homepage](./media/homepage.png)

## Installation
```bash
docker-compose up -d --build
````

## Debug
```bash
docker-compose logs -f
````
![logs](./media/logs.png)

## Stop
```bash
docker-compose down
```
